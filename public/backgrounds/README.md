# 背景プリセット画像の置き場所

ここにドット絵の背景画像を置くと、おへやの背景プリセットとして表示されます。
画像が無いプリセットは、CSSで描いたドット風シーンに自動でフォールバックします。

## ファイル名（この名前で保存してください）

| ファイル名 | プリセット | シーン |
|---|---|---|
| `room.png`   | おへや   | 昼の部屋（窓＋日差し・鉢植え・ラグ） |
| `night.png`  | よぞら   | 夜の部屋（窓の月・ランプ） |
| `meadow.png` | くさはら | 草原（青空・花・丘） |
| `sunset.png` | ゆうやけ | 夕暮れ（夕日・丘・川） |

## 推奨仕様

- 形式: PNG、比率 3:2（例 960×640 / 1024×683）
- 地平線（床の境目）は高さ55〜62%あたり
- 中央下はキャラが立つので、物を置かずシンプルに
- ドット絵はそのまま（拡大時も `image-rendering: pixelated` でくっきり表示されます）

4枚を1枚にまとめた画像（2×2グリッド）で生成した場合は、4分割してそれぞれ上記の名前で保存してください。
