# Buddy — 会話と思い出で育つAIペット

動作するMVPです。会話、経験値・親密度、個性、重要な思い出、Buddy自身の日記をブラウザ内に保存します。

## 起動

```bash
npm install
cp .env.example .env.local
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

## OpenAI連携

`.env.local` に以下を設定します。

```env
OPENAI_API_KEY=あなたのAPIキー
OPENAI_MODEL=gpt-4.1-mini
```

APIキーが空の場合でも、ルールベースのデモモードで一通り動作します。APIキーはブラウザへ送られず、Next.jsのサーバールートだけで利用されます。API呼び出しに失敗した場合も、自動的にデモモードへフォールバックして会話を続けられます。

## 主な機能

- **会話** — ChatGPT (OpenAI Responses API) がBuddyとして応答し、気分・経験値・親密度・個性が変化します
- **思い出の記憶と管理** — 重要な出来事だけをBuddyが選んで記憶。思い出タブから **お気に入り固定・編集・削除** ができ、固定した思い出は会話で優先的に参照されます
- **日記** — 会話ごとの一言に加えて、「今日の日記を書いてもらう」ボタンでChatGPTがその日の会話全体からBuddy視点の日記を書きます
- **育つ個性** — 話す内容にあわせて5つの個性(音楽・運動・知識・優しさ・好奇心)が **増減** し、いちばん高い個性がBuddyの口調ににじみます
- **毎日開きたくなる仕掛け** — 連続来訪ストリーク(🔥)、時間帯であいさつが変化、再訪時にBuddyが思い出を振り返って話しかけます
- **成長・進化の演出** — レベルアップ/進化のお祝いモーダル、5段階の成長段階(称号)、一緒に過ごした日数の表示
- **データの保全・移行** — 成長タブから思い出データをJSONで **バックアップ書き出し／復元** できます
- **背景のカスタマイズ** — おへやの 🖼 ボタンから、ドット風プリセット(おへや／よぞら／くさはら／ゆうやけ)や **好きな画像** を背景に設定できます。画像は端末内に保存され、GitHubには送られません
- **命名・誕生演出** — 初回起動時に名前をつけると、Buddyが目を覚まします

## キャラクターの差し替え

Buddyの見た目は `components/PetAvatar.tsx` に独立しています。`mood`(気分)と `level`(レベル)を受け取って表示するだけなので、別途生成したキャラクター(ドット絵など)に差し替えやすい構造です。気分は `normal / happy / thinking / sleepy / lonely / surprised` の6種類です。

背景は `components/RoomBackground.tsx` に独立しており、ドット風プリセットの追加や差し替えもここで行えます。

## 現在の保存方式

`localStorage` を利用しています(長期利用を想定し、リセット機能は廃止。代わりにバックアップ書き出し／復元を用意)。本番化ではSupabase/PostgreSQLへ移行し、認証、複数端末同期、ベクトル検索による長期記憶を追加してください。
